# Changelog

All notable changes to this repository are documented here.

## Unreleased

### Added

- Added a Tailwind CSS v4 and shadcn/ui design-system stack to `docs-site`.
- Added shadcn-owned command search with `Cmd/Ctrl+K` support for PRD chapters,
  appendices, and diagrams.
- Added responsive global typography tokens for docs prose, navigation, and
  headings.

### Changed

- Restyled the docs site toward a Vercel-docs-inspired interface using neutral
  light/dark tokens, Geist typography, bordered surfaces, and Tailwind
  composition.
- Updated sidebar labels to keep section grouping while hiding leading numeric
  chapter prefixes in navigation labels.
- Updated Mermaid rendering colors to use the neutral docs design tokens.
- Documented docs-site architecture, design-system ownership, and maintenance
  conventions in repo documentation.
